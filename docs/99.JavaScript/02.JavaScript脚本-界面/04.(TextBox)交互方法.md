# (TextBox)交互方法
针对 TextBox 控件，可以通过脚本调用以下方法来控制文本选择。

## 🤖 API 详解与脚本示例

```javascript
// 设置光标选择范围从 start 到 stop
textBox.setSelection(start, stop);

// 将光标移动到指定索引位置
textBox.setSelection(index);

// 选中输入框内的所有文本
textBox.selectAll();

// 从当前光标位置扩展选择范围到指定索引
textBox.extendSelection(index);
```
