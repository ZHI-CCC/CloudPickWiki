# 窗口信息更改

## 💡 功能简介

该功能允许您深度自定义客户端的游戏窗口，包括修改窗口的标题、图标、鼠标指针样式，甚至调整窗口的风格和颜色。

一个显著的特点是，此配置既可以放置在服务端，让玩家进入服务器时生效；也可以放置在客户端本地，在游戏启动时立即生效，提供了极大的灵活性。

## 📁 配置文件

所有窗口信息相关的设置均在 `config/WindowConfig.yml` 文件中进行配置。

## ⚙️ 配置示例

> 💡 以下配置项的功能都比较直观，这里仅做关键注释。如果您有任何疑问，欢迎在官方群中咨询。

```yaml
# 此配置文件可放置于服务端或客户端。
# - 放于服务端: 玩家进入服务器后，窗口样式将发生改变。
# - 放于客户端: 玩家一启动游戏，窗口样式就会应用。

# 窗口标题调整
# 支持使用变量: {loader_type} 会被替换为加载器类型 (如 Fabric), {game_version} 会被替换为游戏版本。
title: 云拾图形-{loader_type}-{game_version}

# 窗口图标设置
# 您可以提供一个或多个不同尺寸的图标，系统会自动根据屏幕DPI选择最合适的那个。
icon:
# - texture/icon-16x16.png
# - texture/icon-32x32.png
# - texture/icon-48x48.png

# 窗口鼠标指针自定义
cursor:
  # 是否启用自定义鼠标指针功能
  enable: true
  # 自定义鼠标指针贴图的宽度和高度
  width: 32
  height: 32
  # 鼠标指针的热点 (Hotspot) 坐标，即实际的点击响应点。
  # 原点 (0,0) 位于贴图的左下角。
  xHotspot: 0
  yHotspot: 32 # 对于一个标准的箭头指针，热点通常在箭头的顶端。
  # 默认状态下的鼠标指针贴图路径 (注意：图片尺寸需为16的倍数，暂不支持网络链接)
  defaultPath: texture/cursor.png
  # 点击状态下的鼠标指针贴图路径 (同上)
  clickPath: texture/cursor_click.png

# 窗口样式调整
# 可选值: NORMAL, FULLSCREEN, FULLSCREEN_BORDERLESS, MAXIMIZED, MAXIMIZED_BORDERLESS, WINDOWED, WINDOWED_BORDERLESS
style: NORMAL

# 窗口尺寸调整 (宽度,高度)
size: 1920,1080

# 是否开启窗口透明背景
# 开启后，游戏世界外的UI背景将变为透明，可以看到桌面。
transparentBackground: true

# 是否同步操作系统的主题 (亮色/暗色模式)
syncSystemTheme: true

# 窗口标题栏颜色 (支持颜色名如 Red, 或十六进制代码如 0xFFFF0000, #FFFF0000)
titleBarColor: Red

# 窗口标题文字颜色
titleColor: Red

# 是否强制开启窗口黑暗模式
darkMode: false

# 窗口材质 (例如 Windows 11 的 Mica 毛玻璃效果)
material: Mica

# 窗口边框颜色
borderColor: Red

# 窗口背景设置
# 可以指向另一个 .yml 文件，用于配置更复杂的背景，例如视频背景。与透明背景配合效果更佳。
background: background.yml
```

## 📝 参数详解

*   `title`: 设置窗口顶部的标题文本。可以使用 `{loader_type}` 和 `{game_version}` 等变量。
*   `icon`: 定义窗口的图标。建议提供 16x16, 32x32, 48x48 等多种尺寸的 `.png` 文件，以适应不同的显示缩放比例。
*   `cursor`:
    *   `enable`: `true` 为启用自定义鼠标指针，`false` 为禁用。
    *   `width`, `height`: 自定义指针图片文件的像素尺寸。
    *   `xHotspot`, `yHotspot`: 定义指针的“有效点击点”。原点在图片的左下角。例如，一个 32x32 的标准箭头，其尖端可能位于 `(0, 31)` 或 `(1, 32)` 附近。
    *   `defaultPath`, `clickPath`: 分别指向常规状态和点击状态下的指针图片，目前仅支持本地路径。
*   `style`: 控制窗口的显示模式。
    *   `NORMAL` / `WINDOWED`: 普通窗口模式。
    *   `FULLSCREEN`: 独占全屏模式。
    *   `FULLSCREEN_BORDERLESS`: 无边框全屏模式（窗口化全屏）。
    *   `MAXIMIZED`: 最大化窗口模式。
*   `size`: 设置窗口的默认分辨率，格式为 `宽度,高度`。
*   `transparentBackground`: 设置为 `true` 后，窗口将变为透明，可以透视看到桌面或其他窗口，适合配合视频背景等功能使用。
*   `syncSystemTheme`: 若为 `true`，窗口会根据你的操作系统（Windows/macOS）的浅色或深色主题自动调整部分外观。
*   `titleBarColor`, `titleColor`, `borderColor`: 分别用于调整窗口标题栏背景、标题文字和边框的颜色。支持常用颜色名和十六进制颜色码。
*   `material`: 应用特定的系统级窗口视觉效果，例如在 Windows 11 上可以设置为 `Mica` 来获得半透明的毛玻璃质感。
*   `background`: 指向一个专门的背景配置文件，用于实现更高级的背景效果，如动态视频背景。
