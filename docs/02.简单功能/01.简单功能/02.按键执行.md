# 按键执行

## 💡 功能简介

该功能允许您为玩家设置特定的键盘按键，当玩家按下这些按键时，可以触发预设的操作。这些操作可以是代表玩家、管理员或服务器后台执行的命令，甚至是更复杂的脚本，为游戏交互提供了极高的自由度。

## 📁 配置文件

所有按键绑定相关的设置均在 `config/KeyConfig.yml` 文件中进行配置。

## ⚙️ 配置示例

下面我们将通过基础和进阶两个示例，来详细讲解如何配置按键绑定。

### 1. 基础按键执行

基础配置允许您直接将一个按键映射到一条或多条命令。您可以指定命令的执行者。

```yaml
# 基础按键执行 —— 可以设置 管理员/后台/玩家 执行命令
Y: # 按键名，此处为 'Y' 键
  # 当玩家按下 'Y' 键时，将按顺序执行以下指令列表
  # [op] 前缀: 表示该命令由管理员权限 (OP) 执行
  - "[op]say 欢迎使用云拾图形-CloudPick 快来组cp"
  # [console] 前缀: 表示该命令由服务器后台执行
  - "[console]say 欢迎使用云拾图形-CloudPick"
  # 无前缀: 表示该命令由玩家自己执行
  - "say 这个指令不是OP没法用啦"
```

### 2. 进阶按键执行

进阶配置提供了更丰富的功能，如自定义按键名称、冷却时间、执行条件等，并且这些按键绑定会显示在客户端的游戏按键设置中，允许玩家自行修改绑定的键位。

```yaml
# V_release 表示在'V'键被松开时触发，而非按下时
V_release:
  # 分类名称: 在客户端按键设置中，此绑定会被归类到 "CloudPick" 下
  category: CloudPick
  # 按键名称: 在客户端按键设置中显示为 "测试执行"
  name: 测试执行
  # 冷却时间: 单位为毫秒，此处设置为 100 毫秒
  cooldown: 100
  # 冷却提示: 在冷却期间尝试触发时，向玩家发送 "冷却中" 的消息
  cooldownMessage: "冷却中"
  # 执行条件列表: 只有满足所有条件时，命令才会被执行
  conditions:
    # 示例条件，格式为: <PlaceholderAPI变量>,<操作符><值>,<不满足条件时的提示信息>
    # 下例表示: 玩家的等级 (%player_level%) 必须大于等于 (>=) 10 级，否则提示 "等级不足十级"
    - "%player_level%,>=10,等级不足十级"
  # 执行的命令列表: 当所有条件满足时执行
  commands:
    - "[op]say 欢迎使用云拾图形-CloudPick"
```

## ⌨️ 按键名大全

以下是您可以在配置文件中使用的标准按键名称列表。

### 🖱️ 鼠标按键

```plaintext
MOUSE_LEFT       # 鼠标左键（主按键）
MOUSE_RIGHT      # 鼠标右键（上下文菜单/副按键）
MOUSE_MIDDLE     # 鼠标中键（滚轮按键）
MOUSE_X1         # 鼠标侧键1（前进键，不同设备可能有差异）
MOUSE_X2         # 鼠标侧键2（后退键，不同设备可能有差异）
```

### ⌨️ 主键盘区

```plaintext
ESCAPE           # ESC键（退出/取消）
1-0              # 数字键1到0（主键盘区上部）
MINUS            # 减号键（-_）
EQUALS           # 等号键（=+）
BACK             # Backspace键（退格）
TAB              # 制表键（缩进/焦点切换）
Q-P              # 字母键Q到P
LBRACKET         # 左方括号键（[{）
RBRACKET         # 右方括号键（]}）
RETURN           # Enter键（回车确认）
LCONTROL         # 左Ctrl键（控制组合键）
A-L              # 字母键A到L
SEMICOLON        # 分号键（;:）
APOSTROPHE       # 单引号键（'"）
GRAVE            # 反引号键（`~）
LSHIFT           # 左Shift键（大写/上档）
BACKSLASH        # 反斜杠键（\|）
Z-M              # 字母键Z到M
COMMA            # 逗号键（,<）
PERIOD           # 句号键（.>）
SLASH            # 斜杠键（/?）
RSHIFT           # 右Shift键
SPACE            # 空格键
CAPITAL          # Caps Lock键（大写锁定）
```

### ⌨️ 功能键区

```plaintext
F1-F24           # 功能键F1到F24（F13-F24常见于专业键盘）
```

### ⌨️ 数字小键盘

```plaintext
NUMLOCK          # 数字锁定键
NUMPAD0-9        # 小键盘数字键
NUMPADENTER      # 小键盘回车键
ADD              # 小键盘加号（+）
SUBTRACT         # 小键盘减号（-）
MULTIPLY         # 小键盘乘号（*）
DIVIDE           # 小键盘除号（/）
DECIMAL          # 小键盘小数点（.）
NUMPADEQUALS     # 小键盘等号（=，某些键盘专用）
```

### ⌨️ 编辑区

```plaintext
INSERT           # Insert键（插入模式）
DELETE           # Delete键（删除）
HOME             # Home键（行首）
END              # End键（行尾）
PAGE_UP          # Page Up键（上一页）
PAGE_DOWN        # Page Down键（下一页）
```

### ⌨️ 方向键

```plaintext
UP               # 上方向键
DOWN             # 下方向键

RIGHT            # 右方向键
```

### ⌨️ 系统控制键

```plaintext
PRINTSCREEN      # Print Screen键（截图）
```

