# 自定义物品渲染

## 💡 功能简介
该功能允许您完全自定义物品的视觉渲染效果，支持使用3D模型或静态/动态图片来替换原版物品外观。

## 📁 配置文件
所有物品渲染自定义设置均在 `config/ItemConfig.yml` 文件中进行配置。

## ⚙️ 功能详解

### 核心特性
- **多格式支持**：兼容模型文件（Blockbench）和图片格式（GIF/PNG/JPG）
- **精准匹配**：通过物品名称、Lore描述或NBT数据识别目标物品
- **动态效果**：支持GIF动图实现动态物品外观
- **精细调整**：提供缩放、偏移、边框等多种视觉参数

## 🎨 配置示例

```yaml
示例物品:
  # 物品识别条件
  match: 测试材质2                    # 匹配物品名称/Lore/NBT
  itemType: sword_diamond            # [可选] 精准匹配物品类型
  
  # 渲染资源配置
  texture: model/aa.png              # 物品贴图（支持GIF/PNG/JPG）
  model: model/model.json            # [模型专用] 基岩版模型文件
  animation: model/animation.json    # [模型专用] 动画配置文件
  
  # 图层效果
  background: texture/传送.png        # 背景图层
  foreground: texture/gif.gif        # 前景图层（支持动态）
  
  # 视觉调整参数
  scale: 1                           # 整体缩放比例
  offsetX: 0                         # 水平偏移量
  offsetY: 0                         # 垂直偏移量
  
  # 边框设置
  border: 0                          # 整体边框大小
  backgroundBorder: 0                # 背景图层边框
  foregroundBorder: 0                # 前景图层边框
  
  # 模型专用调整
  transform: model/xxx.json          # 模型变换配置文件
```

## 🛠️ 参数详解

### 识别参数
| 参数 | 类型 | 说明 |
|------|------|------|
| `match` | 字符串 | 匹配物品名称、Lore或NBT中的关键词 |
| `itemType` | 字符串 | [可选] 精确匹配特定物品类型 |

### 资源参数
| 参数 | 类型 | 说明 |
|------|------|------|
| `texture` | 文件路径 | 主要贴图文件，支持静态和动态图片 |
| `model` | 文件路径 | 3D模型定义文件（基岩版格式） |
| `animation` | 文件路径 | 模型动画配置文件 |
| `background` | 文件路径 | 背景图层贴图 |
| `foreground` | 文件路径 | 前景图层贴图，支持GIF动画 |

### 调整参数
| 参数 | 类型 | 说明 |
|------|------|------|
| `scale` | 数值 | 渲染缩放比例（1.0为原始大小） |
| `offsetX` | 数值 | 水平方向偏移像素 |
| `offsetY` | 数值 | 垂直方向偏移像素 |
| `border` | 数值 | 整体边框厚度 |
| `backgroundBorder` | 数值 | 背景图层额外边框 |
| `foregroundBorder` | 数值 | 前景图层额外边框 |
| `transform` | 文件路径 | 模型变换配置文件 |
