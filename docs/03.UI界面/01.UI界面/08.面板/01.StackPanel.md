[`StackPanel`](/api/avalonia/controls/stackpanel) 通过水平或垂直堆叠的方式排列其子控件。通常，您会使用堆叠面板来排列页面上的一小块 UI 区域。

在 `StackPanel` 内部，如果您没有在子控件上设置与堆叠方向垂直的尺寸属性，子控件会拉伸以填满可用空间。例如，在水平方向下，如果没有显式设置 `Height`，子控件会拉伸以填满可用高度。

在堆叠方向上，`StackPanel` 始终会扩展以容纳其所有子控件。

:::tip
`StackPanel` 不支持滚动。如果堆叠的内容可能超出可用空间，请将 `StackPanel` 放在 `ScrollViewer` 中。
:::

## 常用属性

您最常使用的可能是以下这些属性：

| 属性 | 描述 |
| ------------- | ------------------------------------------------------------------------------- |
| [`Orientation`](/api/avalonia/layout/orientation) | 设置堆叠方向。可选 `Horizontal`（水平）或 `Vertical`（垂直，默认值）。 |
| `Spacing` | 在相邻子控件之间创建均匀的间距。 |
| `HorizontalAlignment` | 控制面板本身在其父容器中的水平位置。 |
| `VerticalAlignment` | 控制面板本身在其父容器中的垂直位置。 |

## 示例

以下 XAML 演示了如何创建一个垂直的堆叠面板。结果会显示子控件拉伸以填满宽度，而堆叠面板的总高度等于所有子控件高度之和。

<XamlPreview>

```yaml
!StackPanel
width: 200
children: 
- !Rectangle
  fill: Red
  height: 50
- !Rectangle
  fill: Blue
  height: 50
- !Rectangle
  fill: Green
  height: 50
- !Rectangle
  fill: Orange
  height: 50
```

</XamlPreview>

## 通过代码定义 StackPanel

以下示例演示了如何使用 `StackPanel` 创建一组垂直排列的按钮。如需水平排列，请将 `Orientation` 属性设置为 `Horizontal`。

<Tabs
defaultValue="xaml"
values={[
{ label: 'XAML', value: 'xaml', },
{ label: 'C#', value: 'cs', },
]}
>
<TabItem value="xaml">

```xml
<StackPanel HorizontalAlignment="Center"
            VerticalAlignment="Top"
            Spacing="25">
    <Button Content="按钮 1" />
    <Button Content="按钮 2" />
    <Button Content="按钮 3" />
</StackPanel>
```

</TabItem>
<TabItem value="cs">

```csharp
// 定义 StackPanel
var myStackPanel = new StackPanel();
myStackPanel.horizontalAlign = 'Center';
myStackPanel.verticalAlign = 'Top';
myStackPanel.spacing = 25;

// 定义子内容
Button myButton1 = new Button();
myButton1.Content = "按钮 1";
Button myButton2 = new Button();
myButton2.Content = "按钮 2";
Button myButton3 = new Button();
myButton3.Content = "按钮 3";

// 将子元素添加到父级 StackPanel 中
myStackPanel.Children.Add(myButton1);
myStackPanel.Children.Add(myButton2);
myStackPanel.Children.Add(myButton3);
```

</TabItem>

</Tabs>

## 居中对齐子项

要将所有子项在堆叠方向上的居中，请将 `HorizontalAlignment` 设置为 `Center`：

```xml
<StackPanel HorizontalAlignment="Center" Spacing="8">
    <Button Content="短按钮" />
    <Button Content="一个更长的按钮" />
</StackPanel>
```

## 水平堆叠并设置间距

通过将 `Orientation` 设置为 `Horizontal` 并添加 `Spacing` 值，可以创建一个水平按钮栏：

```yaml
!StackPanel
orientation: Horizontal
spacing: 12
children:
- !Button
  content: 保存
- !Button
  content: 取消
```

## 实践注意事项

- **尺寸行为**：`StackPanel` 不会在堆叠方向上限制子控件的大小，因此每个子控件都能获得它所需的全部空间。如果您需要子控件按比例分配空间，请考虑使用 `Grid` 替代。
- **性能**：对于包含大量项的列表，建议使用支持虚拟化的 `ItemsRepeater` 或 `ListBox`，而不是在 `StackPanel` 中放置大量控件。
- **滚动**：由于 `StackPanel` 会扩展以容纳所有子控件，它本身不会裁剪内容。如果可能出现溢出，请将其放在 `ScrollViewer` 中。
- **反向顺序**：`StackPanel` 本身不支持反向堆叠。如需反转视觉顺序，可以颠倒子元素的顺序，或使用自定义面板。
