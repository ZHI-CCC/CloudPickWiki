# 按键绑定
该功能可以设置玩家在键盘上点击对应按键时执行的操作，也可以使用脚本执行。

> 📁 在 `config/KeyConfig.yml` 中设置

## 配置示例
```yaml
# 基础按键执行 —— 可以设置 管理员/后台/玩家 执行命令
Y: # 按键名
  # 执行的指令
  - "[op]say 欢迎使用云拾图形-CloudPick 快来组cp" # 管理员执行
  - "[console]say 欢迎使用云拾图形-CloudPick" # 后台执行
  - "say 这个指令不是OP没法用啦" # 玩家执行

# 进阶按键执行 —— 可以在客户端设置中修改绑定的按键
V_release:  # V键松开执行
  category: CloudPick      # 显示在客户端的按键的类别
  name: 测试执行           # 显示在客户端的按键的名称
  cooldown: 100            # 冷却时间 (毫秒)
  cooldownMessage: "冷却中" # 冷却时的提示文本
  conditions:              # 执行条件列表
    - "%player_level%,>=10,等级不足十级" # 示例条件, 格式: <PAPI变量>,<操作符><值>,<不满足条件时的提示>
  commands:                # 执行的命令
    - "[op]say 欢迎使用云拾图形-CloudPick"
```

## 按键名大全
### 🖱️ 鼠标按键
```plaintext
MOUSE_LEFT       # 鼠标左键（主按键）
MOUSE_RIGHT      # 鼠标右键（上下文菜单/副按键）
MOUSE_MIDDLE     # 鼠标中键（滚轮按键）
MOUSE_X1         # 鼠标侧键1（前进键，不同设备可能有差异）
MOUSE_X2         # 鼠标侧键2（后退键，不同设备可能有差异）
```
### ⌨️ 主键盘区
```plaintext
ESCAPE           # ESC键（退出/取消）
1-0              # 数字键1到0（主键盘区上部）
MINUS            # 减号键（-_）
EQUALS           # 等号键（=+）
BACK             # Backspace键（退格）
TAB              # 制表键（缩进/焦点切换）
Q-P              # 字母键Q到P
LBRACKET         # 左方括号键（[{）
RBRACKET         # 右方括号键（]}）
RETURN           # Enter键（回车确认）
LCONTROL         # 左Ctrl键（控制组合键）
A-L              # 字母键A到L
SEMICOLON        # 分号键（;:）
APOSTROPHE       # 单引号键（'"）
GRAVE            # 反引号键（`~）
LSHIFT           # 左Shift键（大写/上档）
BACKSLASH        # 反斜杠键（\|）
Z-M              # 字母键Z到M
COMMA            # 逗号键（,<）
PERIOD           # 句号键（.>）
SLASH            # 斜杠键（/?）
RSHIFT           # 右Shift键
SPACE            # 空格键
CAPITAL          # Caps Lock键（大写锁定）
```
### ⌨️ 功能键区
```plaintext
F1-F24           # 功能键F1到F24（F13-F24常见于专业键盘）
```
### ⌨️ 数字小键盘
```plaintext
NUMLOCK          # 数字锁定键
NUMPAD0-9        # 小键盘数字键
NUMPADENTER      # 小键盘回车键
ADD              # 小键盘加号（+）
SUBTRACT         # 小键盘减号（-）
MULTIPLY         # 小键盘乘号（*）
DIVIDE           # 小键盘除号（/）
DECIMAL          # 小键盘小数点（.）
NUMPADEQUALS     # 小键盘等号（=，某些键盘专用）
```
### ⌨️ 编辑区
```plaintext
INSERT           # Insert键（插入模式）
DELETE           # Delete键（删除）
HOME             # Home键（行首）
END              # End键（行尾）
PAGE_UP          # Page Up键（上一页）
PAGE_DOWN        # Page Down键（下一页）
```
### ⌨️ 方向键
```plaintext
UP               # 上方向键
DOWN             # 下方向键
LEFT             # 左方向键
RIGHT            # 右方向键
```
### ⌨️ 系统控制键
```plaintext
PRINTSCREEN      # Print Screen键（截图）
SCROLLLOCK       # Scroll Lock键（滚动锁定）
PAUSE            # Pause/Break键（暂停）
APPLICATION      # 应用键（等效鼠标右键）
POWER            # 电源键（部分键盘）
SLEEP            # 睡眠键
WAKE             # 唤醒键
```
### ⌨️ Windows专用键
```plaintext
LMETA            # 左Windows键（开始菜单）
RMETA            # 右Windows键
CONTEXT          # 上下文菜单键
```
### ⌨️ 特殊功能键
```plaintext
CLEAR            # 清除键（类似NumLock关闭时的5键）
FN               # 笔记本功能键（通常为组合键）
ZOOM             # 缩放键（某些商务键盘）
HELP             # 帮助键
